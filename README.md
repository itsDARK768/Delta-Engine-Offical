# Friday Night Funkin': Delta Engine
## Friday Night Funkin'
**Friday Night Funkin'** is a rhythm game originally made for Ludum Dare 47 "Stuck In a Loop".

## Delta Engine
**Delta Engine** is a mod for Friday Night Funkin', including a bit of engine rework.
